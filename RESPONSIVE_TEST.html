üß™ Testing Checklist

Desktop Testing (1920px, 1440px, 1024px)

  • Navbar menampilkan menu horizontal
  • Cards menampilkan 4 kolom
  • Hero section 2 kolom (text + form)
  • Hamburger menu tersembunyi
  • Search box inline dengan button

Tablet Testing (768px - 991px)

  • Hamburger menu aktif dan berfungsi
  • Menu dropdown/overlay muncul saat diklik
  • Cards menjadi 1 kolom
  • Hero section menjadi 1 kolom
  • Search box tetap responsive
  • Tables horizontal scroll jika perlu

Mobile Testing (320px - 767px)

  • Navbar fixed di atas layar
  • Brand title disembunyikan (hanya logo)
  • Hamburger menu berfungsi dengan sempurna
  • Menu overlay fullscreen
  • Cards stacked vertikal (1 kolom)
  • Font sizes readable tanpa zoom
  • Buttons minimum 44px untuk tap
  • Input fields minimum 44px untuk mudah diketik
  • Modals centered dan responsive
  • Tables dapat di-scroll horizontal
  • Images tidak melebihi container width
  • No horizontal overflow

üéØ Features yang Ditest

1. Navigation

  • Navbar logo dan title responsif
  • Menu hamburger muncul di tablet/mobile
  • Menu toggle berfungsi dengan smooth
  • Active state menunjukkan halaman saat ini
  • Logout link memiliki styling khusus

2. Hero Section

  • Text content readable di semua ukuran
  • Login card responsive dan centered
  • Hero stats flexible dan tidak overflow
  • Background gradient tetap indah

3. Cards & Grid

  • Cards scaling dengan container
  • Grid responsive menggunakan auto-fill/auto-fit
  • Gap/spacing konsisten
  • Cards tidak terlalu besar di desktop
  • Cards tidak terlalu kecil di mobile

4. Forms & Inputs

  • Input fields full-width di mobile
  • Focus states visible
  • Validation messages readable
  • Buttons full-width di mobile
  • Form labels clear dan properly spaced

5. Tables

  • Tables horizontal scroll di mobile
  • Font sizes reduced tapi readable
  • Header sticky jika perlu
  • Rows hover effects works
  • Table wrapper padding konsisten

6. Modals

  • Modal width responsive
  • Modal vertically centered
  • Close button easily tappable
  • Overlay background semi-transparent
  • Modal tidak overflow viewport

üîß Test dengan DevTools

  1. Buka aplikasi di browser Chrome/Firefox
  2. Tekan F12 untuk membuka DevTools
  3. Klik toggle device toolbar Ctrl+Shift+M (Windows) atau Cmd+Shift+M (Mac)
  4. Pilih device dari dropdown (iPhone, iPad, Galaxy, dll)
  5. Atau drag untuk custom width
  6. Tekan landscape button untuk test landscape mode
